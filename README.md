# Proffy
> Proffy web | Next Level Week 2 - Rocketseat.


## Installation

```bash
# Clone this repository
$ git clone https://github.com/hugomatheus/nlw2-web-proffy.git

# Go into the repository
$ cd nlw2-web-proffy

# Install dependencies
$ yarn install

# Start server
$ yarn start
```

## Technologies

* ReactJS
* Axios
